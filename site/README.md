# 🌳 Yggtree Page

[![Vercel](https://img.shields.io/badge/deployed%20on-Vercel-black?logo=vercel)](https://yggtree.com)
[![Next.js](https://img.shields.io/badge/built%20with-Next.js-black?logo=next.js)](https://nextjs.org)

The official landing page for **[yggtree](https://github.com/logbookfordevs/yggdrasil-worktree)** — the interactive CLI for Git worktrees.

Like the mythical world tree connecting realms, this page serves as the gateway to understanding how `yggtree` turns Git worktrees into a first-class, parallel development workflow.

---

## 🚀 Quick Start

### Development

This is a Next.js project. To start the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

```bash
pnpm build
```

---

## 🏗️ Architecture

- **Framework:** [Next.js 15+](https://nextjs.org) (App Router)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com)
- **Animations:** [Motion](https://motion.dev) (formerly Framer Motion)
- **Deployment:** [Vercel](https://vercel.com)

When deploying from the main Yggtree repository, set the Vercel project
**Root Directory** to `site`.

---

## 🌳 About Yggtree

`yggtree` is designed for developers who:
- Work on multiple tasks in parallel.
- Use AI agents for exploration.
- Want isolation without duplication.
- Value scripted, repeatable setups.

Check out the main repository: **[github.com/logbookfordevs/yggdrasil-worktree](https://github.com/logbookfordevs/yggdrasil-worktree)**
