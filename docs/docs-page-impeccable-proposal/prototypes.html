Three docs page directions.

These are lightweight visual prototypes for the Yggtree docs page. They compare how much brand expression to keep while making the first workflow decision clearer.

Variant A: Refined Grove Keep the current state, but make it calmer.
Yggdrasil Worktree
StartWorkflowsCommandsSafety

Documentation

Use worktrees without losing the plot.

Install once, then choose the workflow that protects the work already in motion. The page starts with decisions before flags.

$npm install -g yggtree

Start planned work

yggtree create

Branch-backed worktree for reviewable tasks.

Jump to a branch

yggtree wc

Reuse or create the checkout, then enter the shell.

Try risky edits

yggtree create-sandbox

Local experiment with an apply path when it works.

What changes

  • Gold becomes a scarce action signal.
  • Glow stays on command moments only.
  • Examples use full containers instead of side accents.
  • The mythic theme becomes background, not instruction.
